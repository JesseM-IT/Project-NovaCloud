# ☁️ Project-NovaCloud


## 🌎 mission 

 Building a hands-on cloud engineering environment using real hardware, Linux, networking, and cloud technologies.
 The goal of this project is to develop practical experience by designing, deploying, troubleshooting, and documenting infrastructure. 

---

## 🖥️ Nova-Core

**Role:** Main Infrastructure Server 

### Hardware
-Mac Mini

### Operating system
-Ubuntu Server

### Current Services 
- Docker🐳
- Portainer
- SSH Remote Administration
- Tailscale Secure Access 
- Pi-hole DNS Filtering
- Samba File Sharing
